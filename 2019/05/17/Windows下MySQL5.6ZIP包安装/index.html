MySQL5.6 Windows下ZIP包安装

本文为Windows平台下MySQL5.6版本ZIP压缩包安装教程

  2. 解压并修改ini配置文件
  3. 安装MySQL
  4. 启动MySQL服务
  5. 设置MySQL密码
  6. 配置环境变量
  7. 验证登陆

1.安装包准备

3.安装MySQL

第一步，使用管理员权限打开命令提示符，可以使用Windows自带的搜索功能搜索，如下图

第二步，使用cd命令切换到MySQL的bin目录，如下图所示

第三步，输入 mysqld -install 命令安装MySQL（mysqld -remove为卸载）

出现Service successfully installed表示安装成功

4.启动MySQL服务

输入 net start mysql 启动MySQL服务，net stop mysql 是停止MySQL服务

5.登陆并设置MySQL密码

使用 mysqladmin -u root -p password 命令登陆MySQL，回车后提示Enter password，默认无密码直接回车，会要求设置新密码，设置自己的密码即可

6.配置环境变量

复制MySQL的bin目录(本例中为D:\MySQL5.6\mysql-5.6.44-winx64\bin)，将其添加到环境变量中，环境变量打开方式：此电脑->属性->高级系统设置->环境变量->系统变量->path

7.验证登陆

使用管理员身份打开命令提示符，使用命令 mysql -u root -p 命令登陆MySQL，登陆成功界面如下图
